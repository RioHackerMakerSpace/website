# RioHackerSpace.github.io

Website of Rio Hacker Space !

## Como contribuir ?

* Obtenha o HUGO CMS
* Clone este projeto
  ```bash
  git clone https://github.com/RioHackerSpace/website.git
  ```
* Para criar um novo post:
  ```bash
  hugo new post/Titulo.md
  ```
* Atualize o public
  ```bash
  hugo
  ```
* Teste
  ```bash
  hugo server 
  ```
* Publique as alterações
  ```bash
  ./deploy
  ```
