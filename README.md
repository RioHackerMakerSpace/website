# RioHackerSpace.github.io

Website of Rio Hacker Space !

## Como contribuir ?

* Obtenha o HUGO CMS
* Clone este projeto
''' git clone https://github.com/riohs/website.git'''
* Para criar um novo post:
'''hugo new post/Titulo.md'''
* Atualize o public
'''hugo'''
* Teste
''' hugo server '''
* Publique as alterações
./deploy
